# MVP_000_SCOPE (Frozen)

## Purpose
This MVP defines the **Lite plan only**.

The purpose is to validate the following core feelings:
- Fun
- Fit to self
- No failure feeling

This MVP is intentionally designed as an activation-only experience,
not a daily or habit-forming product.

## IN (Lite MVP – Fixed)

- Virtual self registration (simplified)
- Unlimited virtual outfit try-on
- Auto exclusion of unsuitable outfits
- Real product catalog with purchase links
- Only items that "do not fail" are shown

## OUT (Explicitly Excluded)

- Any form of daily or periodic recommendations
- Learning optimization or preference memory
- TPO input
- Weather or context logic
- Closet management
- Physical item delivery
- Habit formation mechanics
- Pro-only features

## Success Definition

If users say:

> "This is fun, feels like me, and I don't fail —  
> but I want this every day."

then this MVP is considered successful.

No feature may be added without updating this document.
