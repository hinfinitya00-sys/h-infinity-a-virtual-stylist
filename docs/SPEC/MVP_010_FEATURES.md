# MVP_010_FEATURES (Lite Only)

This MVP includes **only the following 5 features**.

## Included Features

1. Virtual Self Registration
- Users can create their virtual self.
- This is the entry point of all experiences.

2. Body Shape & Ratio Auto-Adjustment
- The system adjusts proportions automatically.
- The goal is “this feels like me”.

3. Unlimited Virtual Outfit Try-On
- Users can freely try outfits with their virtual self.
- This is the core experience of Lite.

4. Real Brand / Real Item Only (with Purchase Links)
- Only real, purchasable items are shown.
- Purchase links are available for each item.
- No fictional assets.

5. Outfit Tendency Indicator (No Blocking)
- Users can try on **any** outfit category. Nothing is hidden or blocked.
- The system shows a tendency label based on body shape / proportions / face-type signals:
  - “Likely to fit” (high match tendency)
  - “Harder to fit” (low match tendency)
- The goal is not to judge, but to provide decision support.

## Explicitly Excluded (for Standard / Pro)

- Outfit history & accumulation
- Recommendations frequency
- Daily updates
- Learning optimization or preference memory
- Event-specific styling
- TPO input
- Weather or context logic
- Closet management
- Physical item delivery
- Habit formation mechanics
- Pro-only features
