# MVP_010_FEATURES (Lite Only)

This MVP includes **only the following 5 features**.

## Included Features

1. Virtual Self Registration
- Users can create their virtual self.
- This is the entry point of all experiences.

2. Body Shape & Ratio Auto-Adjustment
- The system adjusts proportions automatically.
- The goal is “this feels like me”.

3. Unlimited Virtual Outfit Try-On
- Users can freely try outfits.
- This is the core fun of Lite.

4. Real Brand / Real Item Only
- Only real, purchasable items are shown.
- No fictional assets.

5. Auto Exclusion of Unflattering Items
- Items that clearly do not suit the user are excluded.
- Ensures “no failure” experience even in Lite.

## Explicitly Excluded (for Standard / Pro)

- Outfit history & accumulation
- Recommendations frequency
- Daily updates
- Learning optimization
- Event-specific styling
