# MVP_030_CATALOG (Frozen)

## Purpose
This document defines the **minimum world-class catalog requirements**
to make users feel:
> “This looks exactly like wearing it in real life.”

No item may be included unless it meets this realism threshold.

---

## Core Requirement
Every item must reproduce:
- silhouette
- fabric behavior
- size impression
- color fidelity

When applied to the user’s 3D self,
the visual result must closely match real-world wearing perception.

---

## Mandatory Asset Requirements (Lite MVP)

Each clothing item must include:

### 1) Visual Assets
- Front view
- Side view
- Back view
- Fabric close-up
- Natural wearing posture

### 2) Silhouette Parameters
- shoulder structure
- body width
- length
- sleeve volume
- hem behavior

### 3) Fabric Attributes
- thickness (thin / medium / thick)
- stiffness (soft / medium / hard)
- drape (yes / no)
- structure (structured / relaxed)

### 4) Size Information
- actual measurements or ranges
- generic S/M/L only is insufficient

### 5) Color Fidelity
- minimal color correction
- must reflect real product appearance

---

## Exclusion Rules
Items are excluded if they:
- lack multi-angle visuals
- hide silhouette through posing
- rely on heavy photo editing
- cannot reliably map onto a 3D body

---

## Governance Rule
No catalog requirement may be loosened
without updating this document and committing the change to Git.
